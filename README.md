## Official Resources

- [Lua.org](https://www.lua.org/) – Official Lua website.
- [Lua Reference Manual](https://www.lua.org/manual/5.4/) – Language specification.
- [Lua GitHub Mirror](https://github.com/lua/lua) – Source code for Lua.
- [LuaRocks.org](https://luarocks.org/) – Package repository.

## Lua Runtimes & Implementations

- [Lua 5.x](https://www.lua.org/) – Standard reference implementation.
- [LuaJIT](https://luajit.org/) – High-performance Just-In-Time compiler.
- [PUC-Rio Lua](https://www.lua.org/) – Official Lua distribution.
- [Terra](https://github.com/terralang/terra) – Low-level system programming language embedded in Lua.
- [Fengari](https://github.com/fengari-lua/fengari) – Lua VM that runs in the browser (WebAssembly + JS).
- [eLua](https://github.com/elua/elua) – Lua for microcontrollers.

## Package Management

- [LuaRocks](https://github.com/luarocks/luarocks) – Official package manager for Lua.
- [luapower](http://luapower.com/) – Batteries-included Lua distribution.

## Frameworks & Libraries

- [Penlight](https://github.com/lunarmodules/Penlight) – Utility libraries for Lua.
- [LuaFileSystem](https://github.com/lunarmodules/luafilesystem) – File system operations.
- [luasocket](https://github.com/lunarmodules/luasocket) – Networking support for Lua.
- [lua-cjson](https://github.com/mpx/lua-cjson) – JSON parsing and encoding.
- [serpent](https://github.com/pkulchenko/serpent) – Serialization/deserialization library.
- [busted](https://github.com/Olivine-Labs/busted) – Unit testing framework.
- [moonscript](https://github.com/leafo/moonscript) – Syntactically sweeter language that compiles to Lua.
- [lpeg](https://github.com/lunarmodules/lpeg) – Parsing expression grammars.

## Game Development

- [LÖVE](https://github.com/love2d/love) – 2D game development framework.
- [Defold](https://github.com/defold/defold) – Cross-platform game engine that uses Lua.
- [Solar2D (Corona SDK)](https://github.com/coronalabs/corona) – 2D engine using Lua scripting.
- [Roblox Lua (Luau)](https://github.com/Roblox/luau) – Roblox's fast Lua VM and language extension.
- [Gideros](https://github.com/gideros/gideros) – Mobile game development framework.

## Neovim Ecosystem

- [Neovim](https://github.com/neovim/neovim) – Modern Vim fork with Lua as its configuration language.
- [lazy.nvim](https://github.com/folke/lazy.nvim) – Lightweight plugin manager for Neovim.
- [nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter) – Syntax highlighting and parsing.
- [telescope.nvim](https://github.com/nvim-telescope/telescope.nvim) – Fuzzy finder written in Lua.
- [lualine.nvim](https://github.com/nvim-lualine/lualine.nvim) – Status line plugin.
- [Neorg](https://github.com/nvim-neorg/neorg) – Neovim workspace and note-taking system.

## Robotics & Embedded Systems

- [OpenWRT Lua](https://openwrt.org/) – Embedded router OS using Lua.
- [nodemcu-firmware](https://github.com/nodemcu/nodemcu-firmware) – Lua runtime for ESP8266 microcontrollers.
- [luabot](https://github.com/keplerproject/luabot) – Robotics control experiments in Lua.
- [ROS + Lua bindings](https://github.com/gmacario/roslua) – Experimental Lua bindings for the Robot Operating System.

## Web & Networking

- [OpenResty](https://github.com/openresty/openresty) – High-performance web server using Lua on Nginx.
- [Kong](https://github.com/Kong/kong) – API gateway built on OpenResty and Lua.
- [Lapis](https://github.com/leafo/lapis) – Web framework for OpenResty & LuaJIT.
- [Sailor](https://github.com/sailorproject/sailor) – MVC web framework for Lua.
- [Pegasus](https://github.com/EvandroLG/pegasus.lua) – Simple HTTP server.

## Testing

- [busted](https://github.com/Olivine-Labs/busted) – Lua unit testing.
- [luassert](https://github.com/Olivine-Labs/luassert) – Assertion library for Lua.
- [lunit](https://github.com/mkottman/lunit) – xUnit-style testing framework.
